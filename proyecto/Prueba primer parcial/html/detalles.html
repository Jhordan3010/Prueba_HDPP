Justificaciones

Nombre de usuario = Se usó input text porque debe ingresar su nombre.

Contraseña = Se usó input password por que debe ingresar su contraseña y ocultar los caracteres.

Repetir Contraseña = Se usó input password por que debe ingresar su contraseña y ocultar los caracteres.

Dirección = Se usó input text por que debe ingresar su dirección en texto.

Ciudad = Se usó input text por que debe ingresar la Ciudad.

Provincia = Se usó input text por que debe ingresar la provincia.

Número de Huéspedes = Se uso input number por que debe ingresar un valor numérico.

Servicios = Se uso input checkbox por que se seleccionará las opciones.

Pensión = Se uso input checkbox por que se seleccionará las opciones.

Tipo de Habitación = Se uso input checkbox por que se seleccionará las opciones.

Comentario o Sugerencia = Se uso input textarea para que pueda escribir su comentario con el espacio necesesario.

E-mail contacto = Se uso input emai por que se ingresará un correo electrónico y debe contener @.

Acepto la política de privacidad = Se uso input checkbox por que se seleccionará si acepta la política.

Enviar = Se uso input button por que se presionará el boton enviar.

Limpiar = Se uso input button por que se presionará el boton limpiar
